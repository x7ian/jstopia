# CommandLine: The Mothership

The CommandLine is the mothership of the dev fleet.
It is powerful, fast, and direct.
You will learn it step by step.

<DocBlock anchor="what-is-command-line" kind="explanation" title="What is command line?">
The command line is a text interface for your computer.
You type commands and get precise results.
It is faster than clicking for many tasks.
</DocBlock>

<DocBlock anchor="why-devs-use-cli" kind="rule" title="Why devs use CLI">
CLI is repeatable and scriptable.
You can automate tasks and control projects with speed.
It is the toolbelt of developers.
</DocBlock>

<DocBlock anchor="node-repl-and-running-files" kind="example" title="Node REPL and running files">
The Node REPL lets you test JS instantly.
Running files executes full scripts.
These are useful for quick experiments.
</DocBlock>

<DocBlock anchor="packages-high-level" kind="rule" title="Packages at a high level">
Packages are reusable code shared by the community.
Package managers download and organize them.
You will meet npm later in the journey.
</DocBlock>

<DocBlock anchor="lore-commandline-mothership" kind="example" title="Lore: CommandLine mothership">
In JavaScriptopia, the Browser is your personal ship.
The CommandLine is the mothership that builds and repairs ships.
You do not have to master it today, just learn the controls.
</DocBlock>
